# Just Wanna Find That FUNKY Card

這是一款為YGOPRO玩家重新設計的遊戲王查卡片工具，因為YGOPRO的太老了

### 怎麼用
到release下載放進YGOPRO同個資料夾
## 功能

- **多元關鍵字搜尋**：支援輸入卡名、卡片密碼或效果描述進行搜尋 （可選用 **AND** / **OR** 邏輯進行多關鍵字交集或聯集查詢）

- **卡片類型包含/排除整合**：
  - 左鍵點選：包含該卡片類型 （亮藍色）
  - 右鍵/長按點選：排除該卡片類型 （亮紅色）

- **數值邏輯查詢**：支援對攻擊力、守備力進行運算式查詢（例如：`>=2500`、`=?` 查問號數值、或 `atk=def` 查詢攻守相同的卡片）。

- **LUA 腳本分析器**：
  - **可改變卡片名稱**
  - **接觸融合素材**
  - **特殊勝利條件**
  - **constant.lua過濾器**

- **多功能收藏夾**：使用者可自訂不同的收藏資料夾收藏選擇的卡

- **禁卡表**：參照YGOPRO

- **YGOPRODeck API讀取**

- **Yugipedia / Konami Database 快捷** 

- **本地檔案快捷**：支援一鍵打開本地卡片 LUA 腳本與本地縮圖檔案

- **快速複製**：卡片詳情中的卡名、卡片密碼、效果描述均可直接點擊進行剪貼簿複製

- **雙顯示模式切換**：
  - **縮圖模式**：網格檢視大圖，可選 1~6 行顯示
  - **詳細模式**：表格檢視（參照YGOPRO 但比較陽春）

## 安裝與執行說明 （Python）

### 系統需求
- **Python**：3.10 或以上版本吧
- **作業系統**：Windows （支援 DWM 標題列深色美化）、macOS 或 Linux

**依賴**
```bash
pip install PySide6
```
**運行**
```bash
python main.py
```

執行前請確認專案根目錄下有以下必要的資料夾與檔案（或者直接放進YGOPRO同資料夾就行）：
- `main.py`：主程式
- `cards.cdb`：資料庫

選填:
- `lflist.conf`：禁卡表
- `strings.conf`：字段對應
- `script/`：存放腳本
- `pics/`：存放卡圖
